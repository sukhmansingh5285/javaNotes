## ðŸŸ¢ Phase 1: Installation & Setup
**Goal:** Get Maven running on your machine.

1. **`apache-maven-3.5.0`**
	- This is the Maven software. Place it in a permanent location (e.g., `C:\Program Files\apache-maven-3.5.0`).

	- **Action:**

	  - Open a new terminal and run: `mvn -version` to verify Maven is installed.

## ðŸŸ¡ Phase 2: The Basics (Hello World)
**Goal:** Learn Maven project structure and build a simple Java app.

3. **`HelloJava`**
	- Your first Maven project. Teaches directory layout and `pom.xml` basics.
	- **Action:**
	  1. Read `HelloJava/steps.txt`.
	  2. Open a terminal in the `HelloJava` folder.
	  3. Run: `mvn clean install` (compiles and installs the JAR to your local Maven repo).

## ðŸŸ  Phase 3: Dependency Management
**Goal:** Learn how Maven projects use each other as dependencies.

4. **`UsingOwnJar`**
	- Demonstrates using the JAR from `HelloJava` as a dependency.
	- **Action:**
	  1. Ensure you completed Phase 2.
	  2. Open a terminal in `UsingOwnJar`.
	  3. Run: `mvn clean install` (should succeed if `HelloJava` was installed).

## ðŸ”µ Phase 4: Advanced Dependencies & Databases
**Goal:** Work with 3rd party libraries and databases.

5. **`jdbcwithmaven`**
	- Shows how to add external JARs (like Oracle drivers) to your local Maven repo.

	- Demonstrates connecting to MongoDB using Maven dependencies.
	- **Action:**
	  - Check `MongoDbTest/pom.xml` for MongoDB driver.
	  - Run the example if MongoDB is installed.
